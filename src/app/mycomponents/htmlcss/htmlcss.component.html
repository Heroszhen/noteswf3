Un site web adaptatif (anglais RWD pour responsive web design, conception de sites web adaptatifs selon l'OQLF1) est un site web dont la conception vise, grâce à différents principes et techniques, à offrir une consultation confortable sur des écrans de tailles très différentes.

La @mediarègle est utilisée dans les requêtes des médias pour appliquer différents styles pour différents types médias / appareils.
