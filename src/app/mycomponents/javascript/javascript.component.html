Les séléctions

1.first() trouve le premier élément correspondant au sélécteur dans le dom.
2.eq(n) sélectionne un élément ayant un numéro d'index spécifique.
3.last() trouve le dernier élément correspondant au sélécteur dans le dom.
4.children() renvoie tous les éléments qui sont les enfants directs.
5.find () renvoie les descendants des éléments de l'élément sélectionné.
6.prev() renvoie l'élément suivant de l'élément sélectionné.
7.next() renvoie l'élément précédent de l'élément sélectionné.
8.siblings() sélectionne tous les éléments qui sont frères et sœurs du « élément » spécifié.
9.non () retourne des éléments qui ne correspondent pas à certains critères.

BehaviorSubject

Un BehaviorSubject a obligatoirement une valeur par défaut. Il sauvegarde la dernière valeur qu'il a émie et l'envoie aux observateurs lors de leur subscribe.
A.ts:
import { BehaviorSubject } from 'rxjs';
courant$ = new BehaviorSubject([]);

courant$.next(...); //ajouter une valeur
courant$.subscribe(data=>{......}); //récupérer la valeur à l'écoute.
courant$.getValeur(); //récupérer la valeur actuelle
